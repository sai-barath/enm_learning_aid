◆ setArrowsOnNavigationButtonsVisible()

void tgui::FileDialogRenderer::setArrowsOnNavigationButtonsVisible ( bool  showArrows)

Changes whether the arrows are shown on the back, forward and up buttons.

Parameters
showArrowsShould an arrow character be displayed on top of the navigation buttons?

The arrows are shown by default. You may want to hide them if you use textures for those buttons which include an icon. The arrows are meant to be used only when the buttons have a generic theme without an icon on them.
